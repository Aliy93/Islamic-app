# **App Name**: HijriSync

## Core Features:

- Gregorian-Hijri Calendar Display: Displays the current Gregorian date alongside the corresponding Hijri date in a clear, visually appealing format.
- Monthly Calendar View: Presents a monthly calendar view, showing both Gregorian and Hijri dates for each day of the month.
- Islamic Events Highlighting: Automatically highlights important Islamic events, such as Ramadan, Eid al-Fitr, Eid al-Adha, and Ashura, directly on the calendar.
- Hijri Date Adjustment: Allows users to manually adjust the Hijri date (+/- 1 day) to account for differing local sightings.
- Backend API for Hijri Dates: Backend endpoint to fetch Hijri dates from the IslamicFinder API, acting as a proxy to keep the API key server-side. Includes basic caching for each month to reduce API calls.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to evoke a sense of calm and tradition.
- Background color: Light grayish-blue (#ECEFF1), a very desaturated shade of indigo, to create a clean, calming backdrop.
- Accent color: Soft gold (#BDBDBD) to highlight important dates and interactive elements.
- Body and headline font: 'Alegreya', a serif font for a literary and slightly formal feel
- Use minimalist, culturally relevant icons for Islamic events and settings controls.
- Mobile-first, responsive layout that adapts to different screen sizes, ensuring usability on all devices.
- Subtle transitions and animations when navigating between months or adjusting settings to enhance user experience.
